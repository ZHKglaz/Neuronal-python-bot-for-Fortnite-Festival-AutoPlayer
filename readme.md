
# AlphaOne | Neuronal-python-bot-for-Fortnite-Festival-AutoPlayer

An python trained neuronal bot for autoplaying any kind of instrument in fortnite Festival


## How to USE

    1. Download Zip file
    2. Extract where ever you want
    3. LAUNCH " VENV_install.bat" (must have) | after that you will see a new folder named venv
    4. Launch "start.bat"
    5. Press Left Shift key

===================================
## IF ANY ERROR IN STEP 3 TO 4 :

    1. LAUNCH " VENV_install.bat" (must have) | after that you will see a new folder named venv
    2. Close , and launch "setup.bat".
